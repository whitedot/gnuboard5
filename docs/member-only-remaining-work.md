# Member-Only QA Checklist

## 사전 준비

- 관리자 계정 준비
- 메일 발송 확인 가능 상태 준비
- 비밀번호 재설정 링크를 확인할 수 있는 메일함 또는 로컬 메일 수신 환경 준비

## 수동 확인 항목

### 1. 회원가입

- 가입 폼 진입
- 약관 동의 및 제출
- 완료 화면 확인

### 2. 로그인 / 로그아웃

- 로그인 성공
- 보호 페이지 접근 확인
- 로그아웃 후 세션 종료 확인

### 3. 이메일 인증

- 인증 메일 발송
- 인증 링크 처리
- 상태 반영 확인
- 가입 축하 메일/관리자 가입 알림 메일이 추가로 발송되지 않는지 확인

### 4. 비밀번호 재설정

- 찾기 요청
- 재설정 링크 확인
- 신규 비밀번호 로그인 확인

### 5. 회원정보 수정

- 기본 정보 저장
- 비밀번호 변경
- 저장값 반영 확인

### 6. 회원탈퇴

- 탈퇴 절차 수행
- 세션 종료
- 재로그인 불가 확인

### 7. 관리자 회원 관리

- 생성
- 수정
- 삭제 또는 탈퇴 처리

### 8. 관리자 회원 목록
- 회원 목록 페이징 확인
- `cf_write_pages` 제거 후에도 목록 이동이 정상인지 확인

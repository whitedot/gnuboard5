# Member-Only QA Checklist

## 사전 준비

- 관리자 계정 준비
- 메일 발송 확인 가능 상태 준비
- 비밀번호 재설정 링크를 확인할 수 있는 메일함 또는 로컬 메일 수신 환경 준비

## 수동 확인 항목

### 1. 회원가입

- 가입 폼 진입
- 약관 동의 및 제출
- 완료 화면 확인

### 2. 로그인 / 로그아웃

- 로그인 성공
- 보호 페이지 접근 확인
- 로그아웃 후 세션 종료 확인

### 3. 이메일 인증

- 인증 메일 발송
- 인증 링크 처리
- 상태 반영 확인
- 가입 축하 메일/관리자 가입 알림 메일이 추가로 발송되지 않는지 확인

### 4. 비밀번호 재설정

- 찾기 요청
- 재설정 링크 확인
- 신규 비밀번호 로그인 확인

### 5. 회원정보 수정

- 기본 정보 저장
- 비밀번호 변경
- 저장값 반영 확인

### 6. 회원탈퇴

- 탈퇴 절차 수행
- 세션 종료
- 재로그인 불가 확인

### 7. 관리자 회원 관리

- 생성
- 수정
- 삭제 또는 탈퇴 처리

### 8. 관리자 회원 목록
- 회원 목록 페이징 확인
- `cf_write_pages` 제거 후에도 목록 이동이 정상인지 확인

## 2026-03-18 반영 사항

- 완료: 비밀번호 재설정 입력 검증 불일치 수정
- 완료: 가입 운영 메일 설정/발송 제거, 이메일 인증 메일만 유지
- 완료: 관리자 목록의 `cf_write_pages` 참조 제거
- 완료: 관리자 회원 수정 시 관리 메모 저장 경로 복구
- 완료: 이메일 인증 메일 템플릿 문구를 실제 인증 흐름 기준으로 정리
- 완료: 본인인증 변경이력 테이블 키 확인을 실제 설정 키 기준으로 보정
- 남음: 실제 메일 송수신 포함 수동 시나리오 실행 및 결과 기록

## 기록 항목

- 테스트 일시
- 브랜치 / 커밋 해시
- 실패 단계
- 실제 결과
- 재현 조건

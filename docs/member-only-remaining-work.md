# Member-Only QA Checklist

## 사전 준비

- 관리자 계정 준비
- 메일 발송 확인 가능 상태 준비
- 소셜 로그인 제공자 최소 1개 설정

## 수동 확인 항목

### 1. 회원가입

- 가입 폼 진입
- 약관 동의 및 제출
- 완료 화면 확인

### 2. 로그인 / 로그아웃

- 로그인 성공
- 보호 페이지 접근 확인
- 로그아웃 후 세션 종료 확인

### 3. 이메일 인증

- 인증 메일 발송
- 인증 링크 처리
- 상태 반영 확인

### 4. 비밀번호 재설정

- 찾기 요청
- 재설정 링크 확인
- 신규 비밀번호 로그인 확인

### 5. 회원정보 수정

- 기본 정보 저장
- 비밀번호 변경
- 저장값 반영 확인

### 6. 회원탈퇴

- 탈퇴 절차 수행
- 세션 종료
- 재로그인 불가 확인

### 7. 관리자 회원 관리

- 생성
- 수정
- 삭제 또는 탈퇴 처리

### 8. 소셜 로그인

- 최초 연동
- 재로그인
- 기존 계정 매핑 확인

### 9. 콘텐츠 페이지

- `content.php?co_id=company`
- `/content/company` 또는 SEO 주소

## 기록 항목

- 테스트 일시
- 브랜치 / 커밋 해시
- 실패 단계
- 실제 결과
- 재현 조건

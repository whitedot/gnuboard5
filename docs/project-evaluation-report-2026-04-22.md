- README와 현재 범위가 일치하는가:
  - 대체로 일치한다. 현재 저장소는 회원, 인증, 관리자 회원 관리 중심 구조로 정리돼 있다.
- 범위 문서와 실제 파일 구성이 일치하는가:
  - 대체로 일치한다. `lib/bootstrap`, `lib/support`, `lib/domain/member`, `lib/domain/admin` 구조가 실제로 존재한다.
- 제거된 기능 참조가 설정/설치/런타임 경로에 남아 있는가:
  - 평가 시점에는 일부 흔적이 있었지만, 후속 정리로 `homepage`, `device`, `SMS`, `아이핀` 관련 활성 참조를 제거했다.
- 비고:
  - 보고서 작성 직후 같은 작업 세션에서 범위 밖 활성 코드 흔적 1차 정리를 반영했다.

Choices.Js

Choices.js는 가볍고 설정 가능한 셀렉트 박스/텍스트 입력 플러그인입니다. Select2 및 Selectize와 유사하지만 jQuery 의존성이 없습니다.

단일 선택 입력: 기본 (Single Select Input: Default)

기본 단일 선택을 설정하려면 data-choices 속성을 설정하세요.

단일 선택 입력: 옵션 그룹 (Single Select Input: Option Groups)

옵션 그룹을 설정하려면 data-choices data-choices-groups 속성을 설정하세요.

단일 선택 입력: 검색 없음 (Single Select Input: No Search)

검색 기능을 비활성화하려면 data-choices data-choices-search-false data-choices-removeItem 속성을 설정하세요.

단일 선택 입력: 정렬 없음 (Single Select Input: No Sorting)

정렬 기능을 비활성화하려면 data-choices data-choices-sorting-false 속성을 설정하세요.

다중 선택 입력: 기본 (Multiple Select Input: Default)

다중 선택을 활성화하려면 data-choices multiple 속성을 설정하세요.

다중 선택 입력: 삭제 버튼 포함 (Multiple Select Input: With Remove Button)

삭제 버튼을 포함하려면 data-choices data-choices-removeItem multiple 속성을 설정하세요.

다중 선택 입력: 옵션 그룹 (Multiple Select Input: Option Groups)

옵션 그룹이 포함된 다중 선택을 설정하려면 data-choices data-choices-multiple-groups="true" multiple 속성을 설정하세요.

텍스트 입력: 삭제 버튼과 함께 값 제한 (Text Input: Limit Values with Remove Button)

입력 값의 개수를 제한하려면 data-choices data-choices-limit="3" data-choices-removeItem 속성을 설정하세요.

텍스트 입력: 고유 값만 허용 (Text Input: Unique Values Only)

중복되지 않는 고유 값만 허용하려면 data-choices data-choices-text-unique-true 속성을 설정하세요.

텍스트 입력: 비활성화 (Text Input: Disabled)

입력을 비활성화하려면 data-choices data-choices-text-disabled-true 속성을 설정하세요.

Select2

Select2는 표준 셀렉트 박스를 대체하는 고급 플러그인입니다. 검색, 원격 데이터 소스 및 결과의 무한 스크롤을 지원합니다.

참고: 이것은 jQuery 기반 플러그인이므로 작동하려면 jQuery를 포함해야 합니다.
버튼이 있는 단일 선택 입력 (Single Select Input with Button)

Tailwind + Preline을 사용하여 버튼이 추가된 선택 드롭다운 예제입니다.

그룹이 있는 단일 선택 입력 (Single Select Input with Groups)

Select2는 더 나은 구성을 위해 optgroup을 지원하는 일반 셀렉트 박스를 사용할 수 있습니다.

다중 선택 입력 (Multiple Select Input)

그룹화된 옵션과 플레이스홀더를 지원하는 Preline 다중 선택 예제입니다.

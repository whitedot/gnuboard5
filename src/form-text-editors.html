Quilljs

Quill은 호환성과 확장성을 위해 만들어진 최신 WYSIWYG 에디터입니다.

Snow 에디터

Snow는 깔끔하고 플랫한 툴바 테마입니다.

Tailwind로 제작된 강력하고 반응형인 관리자 대시보드 템플릿입니다.


  • 세련되고 현대적인 디자인의 완전 반응형 레이아웃.
  • 로그인, 등록, 대시보드, 차트, 테이블 등 다양한 미리 제작된 페이지.
  • 모달, 경고, 탐색 메뉴 등 다양한 구성 요소 포함.
  • 프로젝트의 필요에 맞게 쉽게 사용자 정의하고 확장 가능.
  • TailwindCSS 4x로 제작되어 다양한 기기와의 호환성 보장.

Summernote

Summernote는 간단하고 사용하기 쉬운 인터페이스로 WYSIWYG 에디터를 만들 수 있게 도와주는 자바스크립트 라이브러리입니다. MIT 라이선스를 따르며 커뮤니티에 의해 유지관리됩니다.

참고: 이것은 jQuery 기반 플러그인이므로 작동하려면 jQuery를 포함해야 합니다.

MyAdmin Admin - Modern Admin Dashboard

MyAdmin Admin은 놀랍고 기능적인 대시보드를 구축할 수 있도록 설계된 강력하고 풍부한 기능의 Tailwind 기반 관리자 템플릿입니다. 데이터 관리 및 통계 제공을 위한 깔끔하고 반응이 빠르며 사용하기 쉬운 인터페이스를 제공합니다. 수많은 구성 요소와 옵션이 있어 모든 유형의 웹 애플리케이션을 구축하는 데 완벽합니다. MyAdmin Admin 사용자 관리에서 차트, 테이블 등에 이르기까지 다음 프로젝트를 시작하는 데 필요한 모든 것을 포함합니다. 성능 및 모바일 반응성에 최적화되어 있어 모든 기기에서 원활한 경험을 보장합니다.

  • 완전 반응형 레이아웃
  • 사용자 정의 가능한 UI 구성 요소
  • Tailwind CSS 4 기반
  • 다양한 바로 사용 가능한 페이지

# Member-Only Refactor

> UI Guide: [docs/ui-kit/](docs/ui-kit/)

## 1. 프로젝트 개요
이 프로젝트는 회원 전용 운영에 필요한 최소 범위만 남기도록 기존 G5 기반 코드를 정리한 저장소입니다.

목표는 두 가지입니다.

- 운영 범위를 계정, 인증, 관리자 회원 관리 중심으로 단순화
- AI 에이전트와 사람이 함께 다루기 쉬운 구조로 공용 라이브러리와 관리자 화면을 정리

## 2. 정리 방향

- 공용 라이브러리 분리
- 관리자 화면 파편화 해소
- 제거 대상 기능의 런타임/설치/문서 의존 축소

## 3. 주요 변경

- `lib/common.lib.php` 계열 기능을 용도별 파일로 분리
- 회원/인증 중심 URL 흐름 정리
- 관리자 설정/회원 관리 화면 정리
- 제거 대상 기능의 설치 스키마와 보조 자산 축소

## 6. 향후 계획

- **UI 프레임워크 현대화 (UI Framework Modernization)**: AI 친화적인 UI/UX 환경 구축 및 개발 생산성 향상을 위해 **Tailwind CSS v4**를 도입하고 적용합니다.
- **명시적 의존성 문서화 (AI-Specific PHPDoc)**: AI 에이전트의 코드 이해도와 문맥 파악 능력을 극대화하기 위한 주석 전략을 도입합니다.
    - **`@global` 태그 활용**: 함수 내부에서 사용되는 전역 변수 의존성을 명시적으로 나열하여 데이터 흐름을 추적 가능하게 합니다.
    - **`@aiagent-description` 태그 도입**: 커스텀 태그를 통해 해당 함수의 역할, 비즈니스 로직, 제약 사항을 AI가 이해하기 쉬운 자연어로 기술합니다.
